# ⬇️ Installation

1. Download the latest **MalumMenu Unofficial** for Steam/Itch from the [Releases](https://github.com/f1xx00rs/MalumMenu-Unofficial/releases/latest) page.
    - File: `MalumMenu-Unofficial-Steam_Itchio.zip`

2. Open the downloaded zip file and copy all its contents.

3. Paste these files directly into your Among Us game folder:
    - **Steam:** Right-click Among Us in Library → `Manage` → `Browse local files`.
    - **Itch.io:** Open the Itch.io app → Right-click Among Us → `Manage` → `Open folder in Explorer`.

4. Launch Among Us as you normally would. A console window will appear to install the mod's requirements. This may take a few minutes during the first launch.

5. After the installation is complete, the game will open with MalumMenu active.
    - By default, you can toggle the cheat GUI by pressing **DELETE**.
